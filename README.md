3️⃣ What is the difference between map(), filter(), and forEach()?
Ans:
এই তিনটিই হলো array method।

map()

এটি array-এর প্রতিটি উপাদানের উপর loop চালিয়ে প্রতিটি উপাদানের উপর operation perform করার জন্য use করা হয়।

এটি নতুন একটি array return করে।

filter()

এটি শর্তসাপেক্ষে array-এর উপাদানের উপর কাজ করে। আলাদা কোনো উপাদান পাওয়া, আলাদা উপাদানের উপর operation perform করা সহ বিভিন্নভাবে কাজ করা যায় এটি দিয়ে।

এটি নতুন একটি array return করে।

forEach()

এটি সাধারণত array-এর সব ক্ষেত্রেই কাজ করে। এছাড়াও এটি বিশেষভাবে NodeList-এর উপর use করা হয়। আর এর কাজ প্রায় map-এর মতোই প্রতিটি উপাদানের উপর কাজ করা।

এটি কোনো কিছু return করে না।
